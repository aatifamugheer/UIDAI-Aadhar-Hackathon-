# 🆔 UIDAI Aadhaar Hackathon

This repository contains my solution for the **UIDAI Aadhaar Hackathon**, where I developed a machine learning pipeline to preprocess data, train models, and evaluate their performance on the provided dataset.

## 🚀 Features
- Data Preprocessing & Cleaning
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Prediction Generation

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📂 Project Structure
```
UIDAI-Aadhaar-Hackathon/
│── data/
│── notebooks/
│── models/
│── results/
│── requirements.txt
│── README.md
```

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/aatifamugheer/UIDAI-Aadhaar-Hackathon.git
```

2. Navigate to the project folder:
```bash
cd UIDAI-Aadhaar-Hackathon
```

3. Install the required packages:
```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run the Jupyter Notebook:

```bash
jupyter notebook
```

Open the notebook and execute the cells to preprocess the data, train the model, and generate predictions.

The project demonstrates an end-to-end machine learning workflow, including data preprocessing, model training, evaluation, and prediction generation.
